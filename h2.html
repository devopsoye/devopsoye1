what is devops: is a software methology or approach that stresses communication, collaboration and integration between software developer and I.T professionals. Devops is a response of the interdependence of software development and I.T operations

Overview of how devops emerged: It began by the common waterfall model which is dev, QA/tester, Operations not talking to each other then comes agile: dev + QA/Tester - Ops DevOps was now Dev + Tester + Ops = DevOps
